# ব্যবহারিক পরীক্ষার নম্বরপত্র — ময়মনসিংহ জিলা স্কুল

চারুকলা ও তথ্য ও যোগাযোগ প্রযুক্তি (ICT) বিষয়ের **ব্যবহারিক / ধারাবাহিক মূল্যায়ন** নম্বর
ইনপুট, হিসাব ও A4 সাইজে প্রিন্টযোগ্য (PDF) নম্বরপত্র তৈরির একটি সহজ অ্যাপ।
শিক্ষক: **মোঃ সারোয়ার আলম জিয়া**, সহকারী শিক্ষক।

## ব্যবহার পদ্ধতি (কোনো ইনস্টল লাগবে না)

1. `index.html` ফাইলটি যেকোনো ব্রাউজারে (Chrome/Edge/Firefox) **ডাবল-ক্লিক** করে খুলুন।
2. **পরীক্ষার তথ্য ও সেটিংস** কার্ডে বিষয়, শ্রেণি, শাখা, সন ইত্যাদি নির্বাচন করুন।
3. **রোল নম্বর যোগ করুন** কার্ডে রোল শুরু ও শেষ দিন (যেমন ১ থেকে ৪০) → “রোল তৈরি করুন”।
   রেঞ্জ দিলে স্বয়ংক্রিয়ভাবে প্রতিটি রোলের সারি তৈরি হবে।
4. টেবিলে প্রতিটি শিক্ষার্থীর **উপস্থিতি (৫), ব্যবহারিক (১০), ভাইভা (১০)** নম্বর দিন।
   মোট, ফলাফল (কৃতকার্য/অকৃতকার্য) ও গ্রেড স্বয়ংক্রিয়ভাবে হিসাব হবে।
5. **🖨 PDF / প্রিন্ট করুন** বাটনে চাপুন → প্রিন্ট ডায়ালগে “Save as PDF” বা প্রিন্টার বেছে নিন।
   পেপার সাইজ **A4**, লেআউট পরিষ্কার ও স্কুলে সাবমিটযোগ্য।

## নম্বর কাঠামো (বিষয়ভেদে আলাদা)

বিষয় নির্বাচন করলে উপযুক্ত স্কিম স্বয়ংক্রিয়ভাবে বসে। চাইলে এক-ক্লিকে প্রিসেট বাটনও আছে।

**🎨 চারুকলা — ধারাবাহিক মূল্যায়ন**

| অংশ | পূর্ণমান |
|---|---|
| ধারাবাহিক মূল্যায়ন | ৫০ |
| **মোট** | **৫০** |
| পাস নম্বর | ১৮ |

**💻 তথ্য ও যোগাযোগ প্রযুক্তি (ICT)**

| অংশ | পূর্ণমান |
|---|---|
| উপস্থিতি | ৫ |
| ব্যবহারিক পরীক্ষা | ১০ |
| ভাইভা | ১০ |
| **মোট** | **২৫** |
| পাস নম্বর | ৯ |

“নম্বর কাঠামো” কার্ড থেকে কম্পোনেন্ট, পূর্ণমান ও পাস নম্বর যেকোনো সময় বদলানো যায় —
তাই নতুন কোনো স্কিম দরকার হলেও কাজ করবে।

## সুবিধাসমূহ

- **রিফ্রেশ দিলেও ডেটা থাকে** — সব তথ্য ব্রাউজারে স্বয়ংক্রিয়ভাবে সংরক্ষিত (localStorage)।
- **একাধিক শীট** — প্রতিটি শ্রেণি/শাখা/বিষয়ের জন্য আলাদা শীট, সহজে অদলবদল।
- **রোল রেঞ্জ থেকে অটো সারি**, অনুপস্থিত চিহ্নিতকরণ, লাইভ সারাংশ।
- **ব্যাকআপ/রিস্টোর** — JSON ফাইল নামিয়ে রাখুন, নতুন ডিভাইসে রিস্টোর করুন।
- **সম্পূর্ণ বাংলায়**, প্রিন্টে বাংলা সংখ্যা।

## টিপস

- ব্রাউজারের ডেটা পরিষ্কার করলে সংরক্ষিত তথ্য মুছে যেতে পারে — তাই মাঝে মাঝে **ব্যাকআপ** নিন।
- সবচেয়ে ভালো প্রিন্টের জন্য Chrome/Edge ব্যবহার করুন এবং প্রিন্ট অপশনে “Background graphics” বন্ধ রাখুন।
