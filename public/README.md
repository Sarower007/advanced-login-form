# public ফোল্ডার

এখানে **`surjo.woff2`** ফন্ট ফাইলটি রাখুন।

`index.html`-এ `@font-face { font-family:'Surjo'; src:url('public/surjo.woff2') ... }`
দিয়ে এই ফন্টটি লোড করা হয়। ফাইলটি এখানে থাকলে পুরো অ্যাপ ও প্রিন্টে সূর্য ফন্ট
ব্যবহার হবে; না থাকলে স্বয়ংক্রিয়ভাবে Hind Siliguri / Noto Sans Bengali ফন্টে
ফলব্যাক করবে — অ্যাপ ভাঙবে না।

> নোট: ফাইলের নাম অবশ্যই হুবহু `surjo.woff2` হতে হবে (ছোট হাতের অক্ষরে)।
